✅ Conceptos aplicados en esta Landing Page:

  • Modelo de Cajas: padding, margin, border en cards y contenedores
  • Box-sizing: border-box en todas las cards para control preciso
  • Display Flex: navegación, hero content, y grid de cards
  • Position Fixed: header y botón de scroll
  • Position Absolute: badge "NUEVO" e íconos de cards
  • Position Relative: contenedores para absolute positioning
  • Z-index: control de capas entre header, badge y botón
  • Display inline-block: botones CTA
