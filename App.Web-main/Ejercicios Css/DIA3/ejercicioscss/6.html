Ejercicio 6: Posicionamiento Static y Relative

📌 Conceptos importantes:

position: static → Posición por defecto, flujo normal del documento

position: relative → Se desplaza DESDE su posición original, pero RESERVA el espacio

Propiedades de desplazamiento: top, right, bottom, left
